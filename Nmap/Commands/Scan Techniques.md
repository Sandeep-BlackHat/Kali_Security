   # How to apply $$ -> Scan Technique
   ## nmap -$$ 'ip'
      Example: nmap -sS 192.168.0.0/24
   
   <table>
    <tr>
      <td><strong>Switch</strong></td>
      <td>Explaination</td>
      </tr>
    <tr>
      <td>-sS</td>
      <td>TCP SYN Scan *Stealth Scan*</td>
    </tr>
    <tr>
      <td>-sT</td>
      <td>TCP connection Port scan</td>
    </tr>
    <tr>
      <td>-sU</td>
      <td>UDP port scan</td>
    </tr>
    <tr>
      <td>-sA</td>
      <td>TCP ACK scan</td>
    </tr>
    <tr>
      <td>-sW</td>
      <td>TCP window scan</td>
    </tr>
    <tr>
      <td>-sM</td>
      <td>TCP Maimon Scan</td>
    </tr>
    <tr>
      <td>-sF</td>
      <td>FIN Scan</td>
    </tr>
  </table>
